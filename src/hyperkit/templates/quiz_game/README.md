# Quiz Game Template

A modern educational quiz starter game made with HyperKit.

## Features

- Multiple-choice quiz flow
- Score and high-score system
- Question progress bar
- Particle feedback for correct answers
- Camera shake feedback for wrong answers
- Input action mapping for answer buttons
- Completion / restart flow
- Responsive virtual canvas scaling

## Run

python main.py

## Gameplay

Read the question and tap one answer button.
Correct answers increase your score.
Finish all questions to complete the quiz.

## Helper Systems Used

This template demonstrates:

- ProgressBar
- ParticleEmitter
- CameraShake
- InputActionMap
- ScoreManager
