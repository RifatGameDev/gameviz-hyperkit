# Generated Project Smoke Tests

Generated project smoke tests make sure HyperKit templates can create real projects successfully.

These tests check that generated projects include:

- main.py
- README.md
- hyperkit.toml
- assets/images
- assets/audio
- assets/fonts
- assets/data

Covered templates:

- tap_counter
- flappy_mini
- swipe_runner
- puzzle_game
- quiz_game
- simple_physics
