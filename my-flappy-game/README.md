# Flappy Mini Template

A modern Flappy-style starter game made with HyperKit.

## Features

- Tap-to-flap gameplay
- Image-based bird and pipe objects
- Auto-generated starter visual assets
- Score and high-score system
- Progress bar target
- Particle feedback
- Camera shake feedback
- Bounds checking
- Input action mapping
- Game over / restart flow
- Responsive virtual canvas scaling

## Run

```bash
python main.py
