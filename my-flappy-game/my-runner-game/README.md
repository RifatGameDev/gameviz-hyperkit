# Swipe Runner Template

A modern lane-based swipe runner starter game made with HyperKit.

## Features

- Lane-based runner movement
- Swipe left/right controls
- Tap left/right side controls
- Image-based player, coin, and hazard objects
- Auto-generated starter visual assets
- Score and high-score system
- Distance progress bar
- Particle feedback
- Camera shake feedback
- Input action mapping
- Game over / restart flow
- Responsive virtual canvas scaling

## Run

```bash
python main.py
